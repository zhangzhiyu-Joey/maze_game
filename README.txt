迷宫游戏
A maze game

玩法说明：
1. 先点击窗口上方的'简单'或'困难'生成迷宫
2. 按键盘方向键控制方块(绿色)移动
3. 单机窗口上方的'答案'即可生成通向终点的路径(红色圆)
